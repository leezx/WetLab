## Methods
- 用 SYBR/EvaGreen 染料法 → 选 qPCR Intercalating Dyes (Primers only)（只要一对引物）。- Power SYBR™ Master Mix
- 用 TaqMan/MGB 探针法 → 选 qPCR (2 Primers + Probe)（一对引物 + 探针）。
